Hydra: a Functional Digital Circuit Description Language

Hydra is a functional language for specifying digital circuits and computer architectures. It uses a functional style to specify circuits clearly and concisely. There are tools to simplify circuit description, including black box abstraction, design patterns, and tools for specialised circuits such as control algorithms. Hydra provides tools for simulating complex circuits.
